# JR Python Bot

Учебный Telegram-бот с викториной и интеграцией ChatGPT.

---

## Стек технологий
- Python 3.11
- aiogram 3
- FSM (Finite State Machine)
- OpenAI API
- python-dotenv

---

## Функциональность
- Диалоговый режим с использованием ChatGPT через OpenAI API
- Имитация общения со знаменитостью на основе системного prompt
- Игровая викторина (quiz) с подсчетом очков и логикой победы
- Управление состояниями пользователя через FSM (aiogram)
- Генерация случайных интересных фактов

---

## Environment variables

Проект использует `.env` для хранения конфиденциальных данных.  
Создайте файл `.env` в корне проекта перед запуском.

| Переменная       | Описание                                      | Обязательная |
|-----------------|-----------------------------------------------|--------------|
| `TOKEN`          | Токен Telegram-бота, полученный у @BotFather | ✅           |
| `ADMIN_ID`       | Telegram ID администратора                     | ❌           |
| `OPENAI_API_KEY` | API-ключ OpenAI                                | ✅           |
| `PROXY`          | HTTP/HTTPS прокси (если требуется)           | ❌           |

**Пример файла `.env.example`:**
```env
TOKEN=
ADMIN_ID=
OPENAI_API_KEY=
PROXY=
